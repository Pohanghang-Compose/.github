# 📚 과제 제출 방식
- 본인 이름으로 레포지토리를 생성합니다.
- feat/week1 등 새 브렌치를 만들고 해당하는 폴더를 생성하여 과제를 진행해주세요. </br>ex) week1 폴더 생성 및 week1 브랜치 생성 후 작업
- PR에 **구현 사진** 혹은 **시연 영상**을 포함하여야 합니다.
- 과제는 다음 스터디 시작일 **하루 전까지** 완료하셔야 합니다. </br>ex) 10월 26일 스터디 진행 -> 25일까지 과제
- 서로서로 리뷰를 달아주며 공부합니다!
